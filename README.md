Sistem Manajemen Kepegawaian

## Deskripsi
Aplikasi untuk mengelola data pegawai, jabatan, absensi, dan laporan.

## Fitur
- Login Admin
- Data Pegawai
- Data Jabatan
- Absensi
- Laporan Pegawai

## Teknologi
- HTML
- CSS
- JavaScript
- PHP
- MySQL

## Cara Menjalankan
1. Clone repository
2. Import database
3. Jalankan pada XAMPP
4. Buka localhost
